# TechPulse

Estructura base de un sitio web de tecnología hecho con HTML semántico.

## Estructura

- `index.html`: página principal sin diseño agregado.
- `pages/`: páginas secundarias del sitio.
- `assets/css/`: estilos futuros.
- `assets/js/`: scripts futuros.
- `assets/img/`: imágenes y recursos visuales.
- `data/`: datos o contenido estructurado para ampliar el sitio.

## Páginas

- Inicio
- Noticias
- Categorías
- Contacto

## Nota

La página principal se dejó sin estilos para respetar tu pedido.
